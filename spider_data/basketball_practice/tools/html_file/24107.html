四种1V1进攻方式

admin 0 条评论 2020-08-31 21:32

    近期很多网友都在关注四种1V1进攻方式相关的话题， 本站也为您准备了一些关于四种1V1进攻方式的相关资讯内容，希望对您有所帮助。
进攻1
 
该练习有助于掌握在葫芦顶处1对1进攻的技巧。防守球员在篮下接过传球以后 迅速向进攻球员方向跑去，开始1对1对峙。
 

 
练习步骤
①进攻球员a和防守球员b在如图所示的位置站立。 b持球。
②b将篮球传给a以后，开始跑向a。
③a接过传球以后迅速做好进攻准备。
 
进攻球员在接球时，轻轻起跳，在 空中接球。从接球到落地为止极短 的时间内，通过观察防守队员的动向，决定一下动作。
 
进攻2
 
这跟上面的“接球进攻” 一样，在对进攻球员更有利的状况下，双方在葫芦顶部 1对1对峙。
 

 
练习步骤
①进攻球员A和防守球员B在如图所示的位置站立。 B持球。
②B将篮球传给A以后，迅速向另一侧的中央区 跑去。
③B到达中央区边线以后，向葫芦顶处跑去，开始 进攻。
 
从进攻球员处于有利位置开始练习1 对1,训练效果会更好。
 
肘区进攻
 
该打法是肘区进攻得分的重要打法。因为该练习中进攻球员处于有利地位，要保证可以得分。
 

 
练习步骤
①进攻球员A和防守球员B在如图所示的位置站立。 B持球。
②B将篮球后旋抛出后，立即向相反方向的罚球线跑去。
③A向对角的肘区跑去，接住B的传球，两人开始1 对1进攻防守。
 
教练笔记：进攻球员在接球时应轻轻起跳，在空中接球。
 
传球1对1进攻
 
在攻防双方均无优势的情况下练习1对1。防守球员给进攻以压力，进攻球员甩开防守，投篮。
 

 
练习步骤
①进攻球员A和防守球员B在如图所示的位置站立。 B持球。
②B将球运到葫芦顶部后把球交给A，随后开始1对 1攻防。
 
教练笔记：在双方谁都不处于有利地位的1对1 对垒下，进攻球员需要更高的水平。 如灵活使用“试探步”等，积极突破防守。
