吉诺比利步法—后仰跳投

admin 0 条评论 2020-08-31 21:40

    近期很多网友都在关注吉诺比利步法—后仰跳投相关的话题， 本站也为您准备了一些关于吉诺比利步法—后仰跳投的相关资讯内容，希望对您有所帮助。
目的：利用“吉诺比利步法”避开防守后，用“跳投”，打法投篮。 为了避免防守队员的进一步阻拦，跳投时可以采取身体后倾的方法。
 

 
练习步骤
 

  

 
①投篮队员A在低位（或中位）持球准备，防守球员B在其身体后侧防守。C或教练在侧翼准备传球。
②A接过C的传球，向罚球线运球。面对B的进一步靠近，用力迈出一大步（吉诺比利步法)。
③如果还没突破B的防守的话，A突然停下，转体（旋转半径要小）投篮。
 

 
练习笔记：在低位或中位区想要运球至篮下投篮，应先观察 对方防守的动作，并加以预测、判断。根据判断， 用力迈出一大步，以迷惑对方；随后突然停下， 再快速转体投篮。能熟练正确使用这一打法，摆脱防守的 可能性就会更高。
 
给球员的建议：“吉诺比利步法”成功的要点之一是“身体接触”，也就是说，在同防守球员身体紧密靠近的状态下转体面向目标。最终大多数球员 会选择难以防守的勾手投篮打法。防守球员会一直阻拦投球，因此掌 握好该打法是非常必要的。
