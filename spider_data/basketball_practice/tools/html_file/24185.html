＂<＂形练习

直立状态下扭动腰的练习。通过这一练习，身体关节可以变得更灵活、柔软，运 动时更有节'奏感。
 

 
练习步骤
①放松直立。
②要注意肩部以上部位没有任何移动，只 是腰部左右移动。
 
刚开始脖子可能会不由自主地和腰一起移动，但要 有意识地控制这一点。有 条件的话，可以在练习时听着音乐， 跟着拍移动。
 
“S ”形练习
 
刚才介绍的“〈”形练习完成后，可以使上半身扭动幅度更大。通过练习，僵硬 的身体关节可以变得更灵活、柔软，但难度系数更大。
 

 
练习步骤
①放松直立。
②肩部向左移动，腰部向右，整体看起来 像“S”形。
 
< ”形练习和“S”形练习最好在教练的口令下有 节奏、连续地进行。刚开 始可能难以跟上，但最好跟着节奏 坚持下去。
